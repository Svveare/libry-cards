# Как добавлять контент в Libry Cards

Весь контент: [`../data/content.json`](../data/content.json)  
Настройки: [`../data/config.json`](../data/config.json)

**Админ (Telegram ID из `config.adminUserIds`)** может править полки/книги/карты в приложении: оверлей в localStorage + экспорт JSON в репо.

```
Stand → Shelf → Book → Page (5) → Card (4 слота)
```

## Правила книги

- **5 страниц**: common → rare → epic → legendary → mythic  
- На странице **4 карточки той же редкости**, что у страницы  
- Secret-карты добавляются отдельно позже  
- В UI все страницы списком (скролл)

| Страница | Редкость |
|----------|----------|
| 1 | Обычная |
| 2 | Редкая |
| 3 | Эпическая |
| 4 | Легендарная |
| 5 | Мифическая |

## Картинки

Рекомендуемый размер: **900×1200** (3:4), WebP/JPG, ~200–400 KB.  
Файлы: `public/cards/` + `"image": "/cards/name.webp"`.
